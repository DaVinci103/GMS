Gap Matrix System #

Introduction #

Gap Matrix System (GMS) is an ordinal notation system created by me (DV103). My main inspiration for this project is Bashicu Matrix System (BMS), in particular, Yto's proof of termination of BMS. BMS is an expansion-based ordinal notation. Terms in BMS are x × y matrices of natural numbers for naturals x and y. I'll write these matrices as BMS(a11,...,a1y)...(ax1,...,axy). Yto shows in the proof a connection between BMS and stable ordinals. Stability is a concept in set theory, defined as follows:

[Definition 1] For positive natural n and ordinals α and β, α is said to be Σn-β-stable iff Lα ≺Σn Lβ.

I.e. for all Σn-sentences ϕ with parameters from Lα, Lα ⊧ ϕ iff Lβ ⊧ ϕ. Let α ≺n β denote that α is Σn-β-stable. Yto showed in her proof in lemma 2.6 that for all positive naturals n, all ordinals α and β so that α is Σn-β-stable and all finite sets X ⊂ β, there is some finite set Y ⊂ α and a mapping f: X → Y so that:

  1. For all x ∈ X ∩ α, x = f(x);
  2. For all x,y ∈ X, x < y iff f(x) < f(y);
  3. For all x,y ∈ X and all positive naturals k, x ≺k y iff f(x) ≺k f(y);
  4. For all x,y ∈ X and all positive naturals m < n, x ≺m β iff f(x) ≺m α.

(The exact lemma she proved is a little different but equivalent.) You might notice that this is similar to patterns of resemblance, but you probably don't because you don't know what that is. One can define a notion similar to that of patterns of resemblance that mimics stability:

[Definition 2] For positive natural n and ordinals α and β, α <n β iff (α, <, <k, <m α)0<k<ω, 0<m<n ≺Σ1 (β, <, <k, <m β)0<k<ω, 0<m<n.

I.e. for all finite sets X ⊂ β, there is a finite set Y ⊂ α so that (Y, <, <k, <m α)0<k<ω, 0<m<n is topologically equivalent to (X, <, <k, <m β)0<k<ω, 0<m<n: there exists a homeomorphism f: X → Y between them. I tried to make transfinite extension of this, which eventually brought me to this ternary relation:

[Definition 3] For ordinals α, β and γ so that α ≤ β ≤ γ, R(α,β,γ) iff (β, α, <, R) ≼Σ1(α) (γ, β, <, R).

You might notice that the relation R(α,β,β+n) for naturals n is similar to α <n+1 β. Terms in my ordinal notation are matrices of numbers with 3 rows so that each column (z,y,x) denotes an ordinal f(z) and a relation R(f(x),f(y),f(z)), and the expansion algorithm is based on reflection principles of R.

Definition #

For matrices X and Y, X + Y denotes the concatenation of X and Y. A term in GMS is an l × 3 matrix M of natural numbers for some natural l with the following properties:

  1. The top row of M is monotonically increasing from 0 (0, 1, 2, ...);
  2. Each column (z,y,x) of M is non-strictly decreasing (x ≤ y ≤ z).

For natural x < l, the xth column of M refers to the unique column with x in the top row. For naturals x,y < l, the part of M from x to y refers to the part of M starting at, including, the xth column up to, excluding, the yth column. For x = y, this refers to the empty matrix. For natural n, the expansion of M with respect to n, denoted M[n], is defined as follows:

Let (c,b,a) be the last column of M. Let A be the part of M from 0 to a, let B be the part of M from a to b and let C be the part of M from b to c. Note that A + B + C is M without its last column. Let D = B if b = c, otherwise, let D = B' + C where B' consists of only the right-most column of B. Let k be the length of D. For each 0 ≤ i < n, let Di be a k × 3 matrix. Di is D with every instance of a number x ≥ b increased by ki + k and every instance of a number x ∈ [a,b) replaced with x + ki + b - a. M[n] is then defined as A + B + C + D0 + ... + Dn-1.

Let o be a mapping from GMS terms to ordinals. For GMS term M, if M is empty then o M := 0, if M is not empty then o M := sup{o M[n] | n < ω}. I coin the ordinal o (0,0,0)(1,1,0)(2,1,0)(3,2,1) as Large. I expect Large to correspond with the PTO of KP0 augmented by the axiom of powerset (not to be confused with KPP0, which has stronger sepparation and comprehension).

GMS terms can be abbreviated. Abbreviations for GMS terms are made by removing duplicate elements from a column, replacing paretheses with spaces, removing the comma's and renaming multi-digit numbers to a single character if needed, e.g. (0,0,0)(1,1,0)(2,1,0)(3,2,1)(4,2,0)(5,5,4)(6,6,6)(7,7,6)(8,7,6)(9,7,6)(10,7,6)(11,11,6) → 0 10 210 321 420 54 6 76 876 976 A76 B6. GMS terms are ordered lexicographically using the abbreviations as described above, e.g. (0,0,0)(1,1,1) < (0,0,0)(1,1,0) as 0 1 < 0 10 lexicographically. For a column (z,y,x) with x ≠ y of a GMS term M, call the parent of (z,y,x) the xth column for y = z and the yth column for y ≠ z. A GMS term M can alternatively be abbreviated by replacing each column (z,y,x) with 1 if x = y and the replacement of its parent plus one, and if y ≠ z, plus the number of ancestors of the xth column, where an ancestor is itself or a parent of another ancestor. For example, (0,0,0)(1,1,0)(2,1,0)(3,2,1)(4,2,0)(5,5,4)(6,6,6)(7,7,6)(8,7,6)(9,7,6)(10,7,6)(11,11,6) → 1,2,4,7,6,7,1,2,4,4,4,2.
