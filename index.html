Gap Matrix System

Introduction

Gap Matrix System (GMS) is an ordinal notation system created by me. My main inspiration for this project is Bashicu Matrix System (BMS), in particular, Yto's proof of termination of BMS. BMS is an expansion-based ordinal notation. Terms in BMS are x × y matrices of natural numbers for naturals x and y. I'll write these matrices as BMS(a11,...,a1y)...(ax1,...,axy). Yto shows in the proof a connection between BMS and stable ordinals. Stability is a concept in set theory, defined as follows:

[Definition 1] For positive natural n and ordinals α and β, α is said to be Σn-β-stable iff Lα ≺Σn β.

I.e. for all Σn-sentences ϕ with parameters from Lα, Lα ⊧ ϕ iff Lβ ⊧ ϕ. Let α ≺n β denote that α is Σn-β-stable. Yto showed in their proof in lemma 2.6 that for all positive naturals n, all ordinals α and β so that α is Σn-β-stable and all finite sets &X; ⊂ β, there is some finite set &Y; ⊂ α and a mapping f: &X; → &Y; so that:

  1. For all x ∈ X ∩ α, x = f(x);
  2. For all x,y ∈ X, x < y iff f(x) < f(y);
  3. For all x,y ∈ X and all positive naturals k, x ≺k y iff f(x) ≺k f(y);
  4. For all x,y ∈ X and all positive naturals m < n, x ≺m β iff f(x) ≺m α.

(The exact lemma they proved is a little different but equivalent.) You might notice that this is similar to patterns of resemblance, but you probably don't because you don't know what that is. One can define a notion similar to that of patterns of resemblance that mimics stability:

[Definition 2] For positive natural n and ordinals α and β, α <n β iff (α, <, <k, <m α)0<k<ω, 0<m<n ≺Σ1 (β, <, <k, <m β)0<k<ω, 0<m<n.

I.e. for all finite sets X ⊂ β, there is a finite set Y ⊂ α so that (Y, <, <k, <m α)0<k<ω, 0<m<n is topologically equivalent to (X, <, <k, <m β)0<k<ω, 0<m<n: there exists a homeomorphism f: X → Y between them. I tried to make transfinite extension of this, which eventually brought me to this ternary relation:

[Definition 3] For ordinals α, β and γ so that α ≤ β ≤ γ, R(α,β,γ) iff (β, α, <, R) ≼Σ1 (γ, β, <, R).

You might notice that the relation R(α,β,β+n) for naturals n is similar to α <n+1 β. Terms in my ordinal notation are matrices of numbers with 3 rows so that each column (z,y,x) denotes an ordinal f(z) and a relation R(f(x),f(y),f(z)), and the expansion algorithm is based on reflection principles of R.
